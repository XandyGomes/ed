# Arrays

Um **array** (vetor) é uma coleção de elementos armazenados em posições de
memória **contíguas**, acessados por um **índice** numérico começando em 0.

## Por que o acesso é O(1)?

Como os elementos ficam lado a lado na memória, o endereço de qualquer
posição pode ser calculado diretamente:

```
endereço(i) = endereço_inicial + i * tamanho_do_elemento
```

Não é preciso percorrer nada — é matemática pura. Por isso `array[i]` é
**O(1)**, tempo constante, independente do tamanho do array.

## Inserção e remoção

Inserir ou remover um elemento **no meio** do array exige deslocar todos os
elementos seguintes uma posição, para manter a contiguidade. Isso é
**O(n)** no pior caso. Inserir ou remover **no final** costuma ser O(1),
já que nenhum outro elemento precisa se mover.
