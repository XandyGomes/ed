# Busca Sequencial

A **busca sequencial** (ou busca linear) é a forma mais simples de procurar
um valor em uma coleção: comparamos o valor procurado com **cada elemento**,
um a um, começando do início, até encontrá-lo ou chegar ao final.

## Algoritmo

```
para i de 0 até n-1:
    se array[i] == alvo:
        retorne i
retorne "não encontrado"
```

## Complexidade

- **Melhor caso:** O(1) — o elemento é o primeiro.
- **Pior caso:** O(n) — o elemento é o último, ou não existe, e precisamos
  examinar todos os n elementos.

## Quando usar?

A grande vantagem da busca sequencial é que ela **não exige que o array
esteja ordenado**. Se os dados não estão ordenados, ou se a coleção é
pequena, a busca sequencial é a escolha mais simples e direta. Para
coleções grandes e ordenadas, a **busca binária** é muito mais eficiente —
veja o próximo tópico.
